# LaTeX Toolbox Gallery

A curated collection of 57 high-quality LaTeX templates, ready to install with [latex-toolbox](https://github.com/thmsgo18/latex-toolbox).

## Installation

```bash
latex-toolbox template install https://github.com/thmsgo18/latex-toolbox-gallery/tree/main/templates/<category>/<template-name>
```

**Example:**

```bash
latex-toolbox template install https://github.com/thmsgo18/latex-toolbox-gallery/tree/main/templates/cv/awesome-cv
```

## Compatibility

Every template in this gallery contains a `main.tex` file at the root of its directory, making it fully compatible with [latex-toolbox](https://github.com/thmsgo18/latex-toolbox).

## License

Each template retains its original license. Please refer to the `LICENSE` file within each template directory. This gallery only includes templates with permissive licenses (MIT, Apache 2.0, LPPL, CC BY/BY-SA/BY-NC-SA).

## Contributing

To suggest a new template, open an issue or pull request. Templates must:
- Have a `main.tex` at the root of their directory
- Be licensed under MIT, Apache, LPPL, or a Creative Commons license
- Compile successfully with a standard LaTeX distribution
