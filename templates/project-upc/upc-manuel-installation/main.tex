\subsection{Objectif du manuel d'installation}

Ce manuel vise à fournir une procédure claire, structurée et détaillée pour
installer correctement l'application. Il s'adresse aux utilisateurs souhaitant
déployer le système dans leur environnement, en garantissant le bon déroulement
de chaque étape.

\begin{remarque}[Prérequis]
  \begin{itemize}
    \item Système d'exploitation : Linux (Ubuntu 22.04+) / macOS / Windows
    \item Python \(\geq\) 3.10 (ou autre langage selon le projet)
    \item Git installé
    \item Accès à Internet
  \end{itemize}
\end{remarque}

\subsubsection{Configuration de Gunicorn (Python/Django)}

\begin{lstlisting}[language=bash, caption=Lancement avec Gunicorn]
gunicorn --workers 3 --bind 0.0.0.0:8000 monprojet.wsgi:application
\end{lstlisting}

\subsubsection{Configuration de Nginx}

\begin{lstlisting}[language=bash, caption=Configuration Nginx (/etc/nginx/sites-available/monprojet)]
server {
    listen 80;
    server_name mondomaine.fr;

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    location /static/ {
        alias /home/user/projet/staticfiles/;
    }
}
\end{lstlisting}

\subsection{Certificat HTTPS}

\begin{lstlisting}[language=bash, caption=Installation de Certbot (Let's Encrypt)]
sudo apt install certbot python3-certbot-nginx
sudo certbot --nginx -d mondomaine.fr
\end{lstlisting}

\subsection{Sécurité serveur}

\begin{itemize}
  \item Configurer le pare-feu (UFW) : autoriser uniquement les ports 22, 80, 443.
  \item Désactiver les connexions SSH par mot de passe (clés SSH uniquement).
  \item Mettre à jour régulièrement le système.
\end{itemize}

\subsection{Vérification}

[Décrire les étapes de vérification post-installation.]

\begin{enumerate}
  \item Ouvrir un navigateur et accéder à \url{http://localhost:8000}
  \item Vérifier que la page d'accueil s'affiche correctement.
  \item Se connecter avec le compte créé précédemment.
  \item Tester les fonctionnalités principales.
\end{enumerate}

\begin{remarque}[Problèmes courants]
  \begin{itemize}
    \item \textbf{Port déjà utilisé :} vérifier qu'aucun autre service n'occupe
      le port 8000 (\texttt{lsof -i :8000}).
    \item \textbf{Erreur de base de données :} vérifier les identifiants dans
      le fichier \texttt{.env}.
    \item \textbf{Fichiers statiques manquants :} relancer
      \texttt{python manage.py collectstatic}.
  \end{itemize}
\end{remarque}
