\section{ElegantNote Instructions}

The ElegantNote template is based on the \LaTeX\ \texttt{article} class. It supports both
\hologo{pdfLaTeX} (recommended for English) and \hologo{XeLaTeX}.

Key features:
\begin{itemize}
  \item Two modes: \texttt{geye} (green background) and \texttt{hazy} (light blue)
  \item Five color themes: \textcolor{eblue}{blue}, \textcolor{egreen}{green},
        \textcolor{ecyan}{cyan}, \textcolor{sakura}{sakura}, black
  \item Customisable font sizes: 8pt -- 20pt
  \item Theorem environments: theorem, lemma, proposition, corollary, definition,
        conjecture, example, remark, note
\end{itemize}

\begin{note}
  Select \lstinline{device=normal} for a standard A4 PDF output.
\end{note}
