\chapter{Problem Definition}

\section{Objective}
The objective of this project is to design and implement a system that:
\begin{itemize}
    \item Processes data efficiently with $O(n \log n)$ complexity
    \item Scales to handle millions of records
    \item Provides a clean API for integration
\end{itemize}

\section{Scope}
The scope includes data ingestion, processing, and output generation. Visualisation is
out of scope for this phase.
