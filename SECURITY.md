# Security Policy

## Scope

This repository contains LaTeX templates and the scripts used to validate,
compile, and package them. Security reports are most relevant for the
[validation/compilation scripts](scripts/) and the
[GitHub Actions workflows](.github/workflows/).

For issues with the templates themselves (for example, an unsafe LaTeX
package or a license problem), see [SOURCES.md](SOURCES.md) and open a
regular issue or pull request.

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Instead, use GitHub's [private vulnerability reporting](https://github.com/thmsgo18/latex-forge-gallery/security/advisories/new),
or email thomas.gourmelen@icloud.com with details and steps to reproduce.

You should expect an initial response within a few days.
